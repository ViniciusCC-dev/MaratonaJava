# Maratona Java

Repositório dedicado ao estudo aprofundado da linguagem Java, cobrindo desde a sintaxe básica até tópicos avançados de programação orientada a objetos e ecossistema Java.

##  Tecnologias e Ferramentas
* **Linguagem:** Java 25+
* **IDE:** IntelliJ IDEA


##  Estrutura do Repositório
* `src/`: Contém todo o código-fonte desenvolvido durante as aulas.


##  Tópicos Estudados
- [x] Tipos Primitivos e Estruturas de Controle
- [ ] Programação Orientada a Objetos (Classes, Objetos, Herança, Polimorfismo)
- [ ] Tratamento de Exceções
- [ ] Coleções (Lists, Maps, Sets)
- [ ] Generics e Lambdas
